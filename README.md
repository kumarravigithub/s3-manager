# Usage

```sh
  npm install -g node-s3-file-manager
  node-file-manager -p 8080
```

Or

```sh
  git clone https://github.com/littleplantstudio/node-s3-file-manager.git
  cd node-s3-file-manager
  npm i
  cd lib
  node --harmony index.js -p 8080
```
